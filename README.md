# Proyecto ABAP - Exercism Track

Este repositorio contiene las soluciones a los ejercicios del track **ABAP** de [Exercism](https://exercism.org/tracks/abap).  
Cada ejercicio está documentado con su descripción, objetivos, reglas y ejemplos de uso.


## Lista de Ejercicios

| Ejercicio | Descripción |
|-----------|-------------|
| **Acronym** | Generar un acrónimo a partir de una frase. |
| **Atbash Cipher** | Implementar el cifrado Atbash. |
| **Armstrong Numbers** | Determinar si un número es Armstrong. |
| **Beer Song** | Generar la canción de "99 Bottles of Beer". |
| **Clock** | Implementar un reloj con operaciones de tiempo. |
| **Collatz Conjecture** | Calcular pasos hasta llegar a 1. |
| **Crypto Square** | Cifrar texto en forma de cuadrícula. |
| **Custom Signs** | Generar mensajes personalizados y coste. |
| **Darts** | Calcular puntuación en una diana. |
| **Difference of Squares** | Diferencia entre suma de cuadrados y cuadrado de la suma. |
| **Eliud's Eggs** | Contar bits activos (huevos). |
| **Elyse's Enchantments** | Manipulación de mazo de cartas. |
| **ETL** | Transformación de estructuras de datos. |
| **Flower Field** | Anotar un tablero con el número de flores adyacentes (tipo Minesweeper). |
| **Grains** | Calcular granos de trigo en un tablero. |
| **Hamming** | Calcular distancia entre cadenas. |
| **High Scores** | Gestión de puntuaciones. |
| **Isogram** | Verificar palabras sin letras repetidas. |
| **Kindergarten Garden** | Asignación de plantas a estudiantes. |
| **Line Up** | Ordenar estudiantes según criterios. |
| **Matrix** | Representación y manipulación de matrices. |
| **Nth Prime** | Calcular el enésimo número primo. |
| **Phone Number** | Validar y formatear números. |
| **Protein Translation** | Traducir RNA a proteínas. |
| **Raindrops** | Convertir números a sonidos según factores. |
| **Resistor Color** | Interpretar colores de resistencias. |
| **Resistor Color Duo** | Leer dos bandas de color. |
| **Resistor Color Trio** | Leer tres bandas de color. |
| **Reverse String** | Invertir cadenas. |
| **Run-Length Encoding** | Compresión y descompresión RLE. |
| **Scrabble Score** | Calcular puntuación de palabras. |
| **Secret Handshake** | Interpretar bits como acciones. |
| **Two-Fer** | Generar mensajes personalizados. |
| **Word Count** | Contar palabras en texto. |
| **RNA Transcription** | Convertir DNA en RNA. |

## Notas Generales

- Cada ejercicio incluye su **propio README** explicativo.
- Se pueden incluir las **clases de test de Exercism (ABAP Unit)** para validar las soluciones.
- Proyecto ideal para practicar lógica, estructuras de datos y manipulación de strings en ABAP.
