# Proyecto ABAP - Exercism Track

Este repositorio contiene las soluciones a los ejercicios del track **ABAP** de [Exercism](https://exercism.org/tracks/abap).  
Cada ejercicio está documentado con su descripción, objetivos, reglas y ejemplos de uso.

## Notas Generales

- Cada ejercicio incluye su **propio README** con una explicación detallada del problema.
- Se incluyen las **clases de prueba de Exercism (ABAP Unit)** para validar las soluciones.
- Proyecto ideal para practicar **ABAP**, algoritmos, estructuras de datos, manipulación de cadenas y tablas internas.
