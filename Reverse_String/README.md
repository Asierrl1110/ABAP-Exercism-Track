# Reverse String

## Descripción

Este ejercicio consiste en **invertir un string**: es decir, devolver la misma cadena de texto pero con sus caracteres en orden inverso. En otras palabras, si la entrada es `"stressed"`, la salida deberá ser `"desserts"`.

## Objetivo

- Crear una función o método en ABAP que reciba una cadena (*string*).
- Devolver esa cadena con los caracteres en orden inverso.

## Instrucciones

1. Define una clase o método llamado, por ejemplo, `reverse_string`.
2. Haz que este método reciba un parámetro de tipo `string`.
3. Internamente, invierte el orden de los caracteres del string.
4. Devuelve la cadena invertida como resultado.

## Ejemplos de uso

| Entrada      | Salida       |
|--------------|--------------|
| `"stressed"` | `"desserts"` |
| `"strops"`   | `"sports"`   |
| `"racecar"`  | `"racecar"`  |
